### 🚀 راه‌اندازی سریع

1. **استخراج فایل‌ها**
   ```bash
   unzip vape-tube-website.zip
   ```

2. **تنظیم پایگاه داده**
   ```bash
   mysql -u root -p
   CREATE DATABASE vape_tube;
   ```

3. **اجرای راه‌اندازی**
   ```bash
   php setup.php
   ```

4. **تست محلی**
   ```bash
   php -S localhost:8000
   ```
